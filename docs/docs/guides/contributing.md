# Contributing

We welcome contributions! Here's how to report issues and request features.

## Reporting Bugs

Found a bug? Help us fix it by reporting it on GitHub.

**[Report a Bug â†’](https://github.com/gopher-lab/gopher/issues/new?template=bug_report.md)**

### Good Bug Reports Include:

- **Clear description** of what went wrong
- **Steps to reproduce** the issue (numbered list)
- **Expected vs. actual behavior**
- **Environment details**: OS, Gopher version, platform (CLI/Desktop)
- **Error messages** or logs from the Activity Log
- **Screenshots** if applicable

The more detail you provide, the faster we can fix it!

## Requesting Features

Have an idea for improving Gopher? We'd love to hear it.

**[Request a Feature â†’](https://github.com/gopher-lab/gopher/issues/new?template=feature_request.md)**

### Good Feature Requests Include:

- **Problem description**: What problem does this solve?
- **Proposed solution**: How should it work?
- **Alternatives considered**: Other approaches you've thought about
- **Use cases**: When would this be helpful?

Feature requests help us prioritize what to build next.

## Getting Help

- **Documentation**: Check the [troubleshooting guide](/guides/troubleshooting) first
- **Existing Issues**: Search [GitHub Issues](https://github.com/gopher-lab/gopher/issues) to see if your issue is already reported
- **Email Support**: [hello@gopher-ai.com](mailto:hello@gopher-ai.com) for direct questions
